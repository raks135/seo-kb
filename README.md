# SEO — Knowledge Base

> **Disclaimer:** This is educational knowledge curation, **not** personalized SEO advice or a guarantee of rankings. Search engines — especially Google — change constantly; treat everything here as a living document and verify against current official guidance (Google Search Central) before acting. Correlation ≠ causation: most "ranking factor" studies show association, not proof of cause.

A version-controlled, source-cited knowledge base covering the full landscape of search-engine optimization:
technical, on-page, off-page/link building, content, keywords, local, international, e-commerce, measurement, algorithms, emerging search, tooling, strategy, and the **traps** that get sites penalized or waste months of effort.

Every non-obvious claim cites ≥2 independent, credible sources (see `STATE/sources.md`).
Three-bucket labeling is used throughout: **robust** · **emerging/unproven** · **folklore/myth**.
